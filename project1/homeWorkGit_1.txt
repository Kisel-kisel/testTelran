First level: 1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделанно предварительно?

-На GitHub: Code-SSH copied;
-В GitBach: git clone ссылка скопированная;

2.Как создать git репозиторий из локального каталога на своем комьютере. 

-В GitBach: git init;
-git add . ;
-git commit -m "new" nameOfFile;

Как его отправить на github.com?

-Coздать репозиторий на GitHub: new-name-create;
-В GitBach: 
git remote add origin git@github.com:Kisel-kisel/qqq.git
git branch -M main
git push -u origin main

3.Как получить на локальный компьютер изменения с github.com.

-В GitBach: git pull;

Дополнительно 



Какую информацию нам дают команда :
git status - показывает состояния файлов в рабочем каталоге и индексе: 
какие файлы изменены, но не добавлены в индекс; 
какие ожидают коммита в индексе. 
Вдобавок к этому выводятся подсказки о том, как изменить состояние файлов.
git log - перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке — последние коммиты находятся вверху.
git remote -v - просмотр файлов скопированных из GitHub.
